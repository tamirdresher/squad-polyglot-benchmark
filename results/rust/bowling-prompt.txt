# Instructions

Score a bowling game.

Bowling is a game where players roll a heavy ball to knock down pins arranged in a triangle.
Write code to keep track of the score of a game of bowling.

## Scoring Bowling

The game consists of 10 frames.
A frame is composed of one or two ball throws with 10 pins standing at frame initialization.
There are three cases for the tabulation of a frame.

- An open frame is where a score of less than 10 is recorded for the frame.
  In this case the score for the frame is the number of pins knocked down.

- A spare is where all ten pins are knocked down by the second throw.
  The total value of a spare is 10 plus the number of pins knocked down in their next throw.

- A strike is where all ten pins are knocked down by the first throw.
  The total value of a strike is 10 plus the number of pins knocked down in the next two throws.
  If a strike is immediately followed by a second strike, then the value of the first strike cannot be determined until the ball is thrown one more time.

Here is a three frame example:

|  Frame 1   |  Frame 2   |     Frame 3      |
| :--------: | :--------: | :--------------: |
| X (strike) | 5/ (spare) | 9 0 (open frame) |

Frame 1 is (10 + 5 + 5) = 20

Frame 2 is (5 + 5 + 9) = 19

Frame 3 is (9 + 0) = 9

This means the current running total is 48.

The tenth frame in the game is a special case.
If someone throws a spare or a strike then they get one or two fill balls respectively.
Fill balls exist to calculate the total of the 10th frame.
Scoring a strike or spare on the fill ball does not give the player more fill balls.
The total value of the 10th frame is the total number of pins knocked down.

For a tenth frame of X1/ (strike and a spare), the total value is 20.

For a tenth frame of XXX (three strikes), the total value is 30.

## Requirements

Write code to keep track of the score of a game of bowling.
It should support two operations:

- `roll(pins : int)` is called each time the player rolls a ball.
  The argument is the number of pins knocked down.
- `score() : int` is called only at the very end of the game.
  It returns the total score for that game.

#[derive(Debug, PartialEq, Eq)]
pub enum Error {
    NotEnoughPinsLeft,
    GameComplete,
}

/// Snapshot of the game's progress derived by replaying the recorded rolls.
struct State {
    /// Whether all ten frames (and any fill balls) have been rolled.
    complete: bool,
    /// Pins still standing for the next throw (only meaningful when not complete).
    pins_available: u16,
}

#[derive(Default)]
pub struct BowlingGame {
    rolls: Vec<u16>,
}

impl BowlingGame {
    pub fn new() -> Self {
        BowlingGame::default()
    }

    pub fn roll(&mut self, pins: u16) -> Result<(), Error> {
        let state = self.analyze();
        if state.complete {
            return Err(Error::GameComplete);
        }
        if pins > state.pins_available {
            return Err(Error::NotEnoughPinsLeft);
        }
        self.rolls.push(pins);
        Ok(())
    }

    pub fn score(&self) -> Option<u16> {
        if !self.analyze().complete {
            return None;
        }
        Some(self.compute_score())
    }

    /// Walk the recorded rolls frame by frame to determine whether the game is
    /// finished and how many pins are available for the next throw.
    fn analyze(&self) -> State {
        let rolls = &self.rolls;
        let mut i = 0;

        for frame in 0..10 {
            if frame < 9 {
                // Frames 1-9.
                match rolls.get(i) {
                    None => return State { complete: false, pins_available: 10 },
                    Some(&10) => {
                        // Strike: frame ends after a single roll.
                        i += 1;
                    }
                    Some(&first) => match rolls.get(i + 1) {
                        None => {
                            return State { complete: false, pins_available: 10 - first }
                        }
                        Some(_) => i += 2,
                    },
                }
            } else {
                // Tenth frame, including any fill balls.
                let first = match rolls.get(i) {
                    None => return State { complete: false, pins_available: 10 },
                    Some(&p) => p,
                };

                if first == 10 {
                    // Strike grants two fill balls.
                    let second = match rolls.get(i + 1) {
                        None => return State { complete: false, pins_available: 10 },
                        Some(&p) => p,
                    };
                    match rolls.get(i + 2) {
                        None => {
                            // Second fill ball: pins reset only if the first fill
                            // ball was itself a strike.
                            let pins_available = if second == 10 { 10 } else { 10 - second };
                            return State { complete: false, pins_available };
                        }
                        Some(_) => return State { complete: true, pins_available: 0 },
                    }
                }

                let second = match rolls.get(i + 1) {
                    None => return State { complete: false, pins_available: 10 - first },
                    Some(&p) => p,
                };

                if first + second == 10 {
                    // Spare grants one fill ball.
                    match rolls.get(i + 2) {
                        None => return State { complete: false, pins_available: 10 },
                        Some(_) => return State { complete: true, pins_available: 0 },
                    }
                }

                // Open tenth frame: done after two rolls.
                return State { complete: true, pins_available: 0 };
            }
        }

        State { complete: true, pins_available: 0 }
    }

    /// Sum the score across all ten frames. Only called once the game is
    /// complete, so every bonus roll referenced here is guaranteed to exist.
    fn compute_score(&self) -> u16 {
        let rolls = &self.rolls;
        let mut total = 0;
        let mut i = 0;

        for _ in 0..10 {
            if rolls[i] == 10 {
                // Strike: ten plus the next two rolls.
                total += 10 + rolls[i + 1] + rolls[i + 2];
                i += 1;
            } else if rolls[i] + rolls[i + 1] == 10 {
                // Spare: ten plus the next roll.
                total += 10 + rolls[i + 2];
                i += 2;
            } else {
                // Open frame: just the two rolls.
                total += rolls[i] + rolls[i + 1];
                i += 2;
            }
        }

        total
    }
}
