# Instructions

```text

+-++--+

+--+--+
```

```text

```

```text


   +--+
   |  |
   +--+
```

```text


+--+
|  |
+--+
```

```text
  ++
  ++


```

You may assume that the input is always a proper rectangle (i.e. the length of every line equals the length of the first line).
